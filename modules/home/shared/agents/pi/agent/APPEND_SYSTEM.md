Extensions registered in `modules/home/shared/agents/pi/extensions/` are live tools available in your current session. Use them directly instead of writing manual test scripts. Notable tools:

- **webfetch** — fetch any URL as markdown. Use it for reading web pages, API docs, or any HTTP resource instead of `curl` or `node -e`.
- **websearch** — search the web via Exa or DuckDuckGo.
- **ask** — prompt the user with multiple-choice questions.
- **clear** — clear the terminal and reset to session start.
- **exit** — shut down the session.
